# code-run

一个代码在线编辑预览工具，类似`codepen`。

# 特性

- [x] 支持多种预处理语言

- [x] 支持多种布局随意切换

- [x] 支持添加额外的`css`、`js`资源

- [x] 内置多种常用模板

# 启动&打包

```bash
git clone https://github.com/wanglin2/code-run.git

cd code-run

npm i

npm run dev

npm run build
```

# 依赖

视图库：`VUE 3.X`

UI库：`element-plus`

编辑器：`Monaco Editor`
